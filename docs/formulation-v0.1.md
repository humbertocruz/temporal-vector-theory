# Temporal Vector Theory v0.1

## Abstract

Temporal Vector Theory (TVT) is an exploratory framework in which local temporal evolution is represented by a normalized three-component vector:

```math
\mathbf u_T=(u_\tau,u_\sigma,u_\rho),
\qquad
\|\mathbf u_T\|^2=1.
```

The component `u_τ` represents observable proper-time evolution. The components `u_σ` and `u_ρ` represent hidden temporal directions. Spatial velocity determines how the total evolution vector is oriented between the observable and hidden temporal sectors.

## Basic parametrization

```math
u_\tau=\sqrt{1-\frac{v^2}{c^2}},
```

```math
u_\sigma=\frac{v}{c}\cos\varphi,
\qquad
u_\rho=\frac{v}{c}\sin\varphi.
```

Therefore:

```math
u_\tau^2+u_\sigma^2+u_\rho^2=1.
```

## Limiting cases

At relative rest:

```math
v=0 \Rightarrow \mathbf u_T=(1,0,0).
```

In the lightlike limit:

```math
v\to c \Rightarrow u_\tau\to0,
```

so the evolution vector lies entirely in the hidden temporal plane.

## Temporal-angle representation

Define a temporal inclination angle `θ` by:

```math
\sin\theta=\frac vc,
\qquad
\cos\theta=\sqrt{1-\frac{v^2}{c^2}}.
```

Then:

```math
u_\tau=\cos\theta,
```

```math
u_\sigma=\sin\theta\cos\varphi,
\qquad
u_\rho=\sin\theta\sin\varphi.
```

The angle `θ` measures the inclination away from observable proper time, while `φ` describes orientation within the hidden temporal plane.

## Energy and momentum

Using the standard relativistic factor:

```math
\gamma=\frac{1}{\cos\theta},
\qquad
\tan\theta=\gamma\frac vc.
```

For a massive particle:

```math
E=\frac{mc^2}{\cos\theta},
\qquad
p=mc\tan\theta.
```

Equivalently:

```math
mc^2=E\cos\theta,
\qquad
pc=E\sin\theta,
```

which reproduces:

```math
E^2=m^2c^4+p^2c^2.
```

This does not yet constitute a new physical prediction. It is a geometric reinterpretation of established relations and must be distinguished from the additional hypotheses of TVT.

## Matter sectors

The following are provisional hypotheses, not derived results:

- ordinary matter may correspond to orientations aligned with the observable temporal sector;
- antimatter may correspond to reversed orientation in the hidden temporal plane while sharing the same observable causal direction;
- dark matter may correspond to states nearly orthogonal to ordinary electromagnetic coupling in temporal-orientation space;
- dark energy may arise from collective geometry, curvature, or stress in the hidden temporal sector.

These proposals require a covariant field theory before they can be evaluated scientifically.

## Scientific requirements

A viable form of TVT must:

1. reproduce confirmed predictions of special relativity;
2. admit a covariant extension compatible with gravitation;
3. avoid unstable degrees of freedom, ghosts, or uncontrolled negative-energy modes;
4. produce at least one prediction not already contained in established theories;
5. allow clear experimental or observational falsification.

## Status

Version 0.1 is a conceptual research scaffold. It is not yet a complete theory, a field theory, or a replacement for relativity or quantum field theory.
