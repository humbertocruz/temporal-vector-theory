# Trigonotime

**Trigonotime** is the provisional mathematical language of Temporal Vector Theory. It studies projections, angles, rotations, and distances between temporal-evolution vectors in a multidimensional temporal space.

## Temporal unit sphere

Let

```math
\mathbf u_T=(u_\tau,u_\sigma,u_\rho)
```

with

```math
u_\tau^2+u_\sigma^2+u_\rho^2=1.
```

The endpoint of `\mathbf u_T` lies on a temporal unit sphere.

## Angular coordinates

```math
u_\tau=\cos\theta,
```

```math
u_\sigma=\sin\theta\cos\varphi,
\qquad
u_\rho=\sin\theta\sin\varphi.
```

Here:

- `θ` is the inclination away from observable proper time;
- `φ` is the orientation within the hidden temporal plane.

## Velocity relation

```math
\sin\theta=\frac vc,
\qquad
\cos\theta=\sqrt{1-\frac{v^2}{c^2}}.
```

Thus:

```math
\tan\theta=\gamma\frac vc=\frac{p}{mc}.
```

## Energy projection

```math
mc^2=E\cos\theta,
\qquad
pc=E\sin\theta.
```

The standard energy-momentum relation follows as the temporal Pythagorean identity:

```math
E^2=m^2c^4+p^2c^2.
```

## Angular separation

For normalized temporal vectors `\mathbf u_A` and `\mathbf u_B`:

```math
\cos\Omega_{AB}=\mathbf u_A\cdot\mathbf u_B.
```

Using angular coordinates:

```math
\cos\Omega_{AB}=\cos\theta_A\cos\theta_B+\sin\theta_A\sin\theta_B\cos(\varphi_A-\varphi_B).
```

A future dynamical theory may use `\Omega_{AB}` to parametrize couplings between temporal sectors. No such coupling law is established yet.

## Temporal rotations

A transformation of temporal orientation may be written as:

```math
\mathbf u_T'=R_T\mathbf u_T,
```

where `R_T` is a rotation or generalized Lorentz-type transformation in temporal state space.

A central unresolved question is whether the correct geometry is circular, hyperbolic, pseudo-Riemannian, or a combination of internal circular geometry and spacetime hyperbolic geometry.

## Scientific caution

Trigonotime currently reorganizes known relativistic relations into an angular language. It becomes new physics only when a consistent dynamics produces novel, falsifiable predictions.
