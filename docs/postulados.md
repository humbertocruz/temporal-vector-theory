# Postulados provisórios

## P1. Estrutura temporal vetorial

O estado temporal local de uma partícula ou observador é descrito por um vetor tridimensional:

```math
\mathbf u_T=(u_\tau,u_\sigma,u_\rho).
```

## P2. Normalização

A magnitude da evolução temporal total é invariante e normalizada:

```math
\mathbf u_T\cdot\mathbf u_T=1.
```

## P3. Tempo observável como projeção

O tempo próprio mensurável é a projeção de `\mathbf u_T` sobre a direção temporal acessível ao observador.

## P4. Redistribuição cinemática

A velocidade espacial relativa altera a orientação do vetor temporal sem alterar sua magnitude total.

```math
u_\tau=\sqrt{1-\frac{v^2}{c^2}}.
```

## P5. Plano temporal oculto

As componentes restantes satisfazem:

```math
u_\sigma^2+u_\rho^2=\frac{v^2}{c^2}.
```

## P6. Orientação interna

Um ângulo interno `\varphi` distribui a componente oculta entre duas direções:

```math
u_\sigma=\frac vc\cos\varphi,
\qquad
u_\rho=\frac vc\sin\varphi.
```

## P7. Limite relativístico

Toda previsão observável da TTV deve reduzir à relatividade especial no regime em que os graus temporais ocultos não produzem acoplamentos adicionais.

## P8. Covariância

A formulação final não pode depender de um referencial absoluto privilegiado. Quantidades físicas devem ser expressas por invariantes, campos ou relações entre observadores.

## P9. Falsificabilidade

A existência de dimensões temporais adicionais só terá significado físico se produzir ao menos um efeito observável distinto das teorias estabelecidas.

## Estado

Esses postulados são provisórios. Eles funcionam como restrições de projeto, não como fatos estabelecidos.
